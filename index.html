> Question / Réponse transverse Génie Log / Web Serveur

Est-ce que je peux faire les TPs chez moi ?

  • Si je n'ai pas de fiche d'appel à vous faire remplir, oui. Sinon, vous pouvez venir, signer, et partir. Compte-tenu des conditions sanitaires actuelles, je n'imposerai pas la présence.
  • Vous pourrez toujours me contacter par mail si vous avez des questions.
  • Cela ne vous dispense évidemment pas de faire le TP.

Est-ce qu'on peut faire les TP à 2 ?

  • Vous pouvez envoyer un rendu/code-source par binôme (ou en solo si vous préférez), pas plus.
  • Par contre, vous pouvez tous vous entraider, aucun soucis.
  • Si vous le faites en binôme, j'apprécierai d'autant plus avoir un dépôt Git, pour m'assurer que vous savez utiliser l'outil.

Si j'envoie un code qui ne compile pas, comment ça se passe ?

  • Dans un premier temps, j'enverrai un mail pour savoir si je ne me suis pas trompé dans la manière de compiler le code.
  • Si le code ne fonctionne pas, je prendrai en compte le code fait, l'implication, les tentatives, etc.
  • En bref : vous n'aurez pas une "mauvaise note" juste car le code ne compile pas.
  • Par contre, il est dans ce cas primordial de me donner le + de choses possibles sur vos tentatives et vos blocages !

On à un/des partiel(s) proche(s) d'un rendu de TP !

  • Dites à vos délégués de m'envoyer un mail ! Je décalerai le rendu pour vous permettre d'étudier vos partiels et de rendre votre TP dans des conditions convenables.
  • Note : Dans la limite de mes capacités, je ne décide pas des dates butoires pour communiquer vos notes à la fac.

Qui effectue la notation ?

  • Moi.

J'ai une ou des questions...

  • Avant toute chose, vérifiez la FAQ du TP correspondant !
  • Envoyez-moi un mail (voir dans les onglets "Génie Logiciel" et "Technologie Web-Serveur" pour les détails).
  • Si vous n'avez pas de réponse sous 48h, relancez-moi (et désolé si ça arrive).
  • Si la question est d'intérêt public, je la rajouterai dans la FAQ de la section concernée.
